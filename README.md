# ALU Unity Project

A Unity project focused on user interface development and implementation.

## Project Structure

- `unity-user_interface/` - Contains Unity UI-related assets and components
  - `0-project_add_new` - Instructions for adding new assets (File > New Asset)

## Getting Started

1. Open the project in Unity
2. Navigate to the unity-user_interface folder for UI components
3. Use File > New Asset to add new project elements

## Requirements

- Unity (version to be specified)
- Compatible development environment

## Contributing

Follow Unity best practices when adding new assets and components to the project.
